## 博客创建 

依靠强大的GPT，终于学会了搭建自己的博客，短暂学会了git，hugo，还有html语言
不得不说现在的AI实在是太强大了

推送博客流程如下：
### 1. 进入 Hugo 项目文件夹

cd D:\WHU\Website\jackypercy-hugo

### 2. 本地预览

hugo server

### 3. 浏览器打开：

http://localhost:1313/
确认首页、About、Publications、Blogs、Search 都没问题。
看完之后，在终端按：
Ctrl + C
退出本地预览。

### 4. 查看改了哪些文件
git status

### 5. 添加所有修改
git add .

### 6. 提交修改
git commit -m "Update website"

### 7. 推送到 GitHub
git push

推送成功后，去 GitHub 仓库：
Actions
